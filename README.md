# Sparkle Clean Services

Sparkle Clean Services is a local cleaning service web app designed to help local citizens
find suitable cleaning services for their homes and offices.

**Live Demo:** [View Site](#)

---

## Features

- Login and Sign-Up Modal
- Book a cleaning using a form
- View detailed information for each cleaning service
- Pricing tier styled table
- Responsive layout for mobile, tablet, and desktop
- Contact using a form
- Reusable navigation and button components
- Consistent typography and spacing system

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap

---

## Pages

- Home
- About
- Services
- Pricing
- Contact
- Booking

## Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Navigate through the pages using the menu links
